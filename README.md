# hotel_des_murmures

# TP – Agilité – Tests unitaires

## Projet BlueJ : L’Hôtel des Murmures

Ce dépôt contient le travail réalisé dans le cadre du **TP d’Agilité – Tests Unitaires (TPJU)** du Master MIDO.

Le projet vise à découvrir, par l’expérimentation, les concepts de la programmation orientée objet en Java ainsi que les pratiques de tests unitaires, à l’aide de l’outil **BlueJ**.

---

## Contenu du dépôt

* `Hotel.java` : classe principale (« classe fétiche ») représentant l’hôtel
* `Repas.java` : classe associée représentant les repas proposés
* `HotelTest.java` / `RepasTest.java` : tests unitaires JUnit
* `package.bluej` : configuration du projet BlueJ
* `Tutoriel_Hotel_des_Murmures.pdf` : tutoriel pédagogique illustré (captures d’écran et commentaires)
* `README.md` : ce fichier

---

## Sujet

Le projet met en scène **un hôtel hanté**, dont le comportement évolue au fil des itérations :

* gestion du nombre de chambres et du tarif par nuit
* rénovation de l’hôtel impactant le prix
* ajout de repas et calcul du prix d’un séjour
* associations entre classes (unidirectionnelles puis bidirectionnelles)
* écriture et exécution de tests unitaires
* refactoring et exécution des tests en ligne de commande

L’approche est volontairement progressive et concrète, sans théorie formelle.

## Ouvrir le projet

1. Cloner le dépôt :

   ```bash
   git clone https://github.com/djenajj/hotel_des_murmures.git
   ```
2. Lancer BlueJ
3. `Project → Open Project`
4. Sélectionner le dossier du projet
